# 🚀 Getting Started

**1. Clone the Repository:**

        git clone https://github.com/ZeeRox-dev/Route-IQ.git
        cd routeiq
   
**2. Run the Application**

Simply open the HTML file in your browser:

open Route_IQ_V5.html

or double-click the file.

# 🔑 HERE API Setup (Optional)

**To enable live traffic data:**

1. Go to: [https://developer.here.com](https://www.here.com/developer)
2. Create an account and get your API key
3. Enter the key in the app when prompted

The key is stored locally using localStorage.
